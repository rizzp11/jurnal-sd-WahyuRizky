# Mid-Term Reflection - Software Development (Minggu 8)

## 1. Materi yang Paling Sulit Dikuasai (Minggu 1-7)

Selama paruh pertama semester, materi yang paling menantang bagi saya adalah manajemen percabangan Git (Branching & Merging) untuk menghindari konflik serta penerapan arsitektur MVC yang disiplin saat mulai memisahkan komponen kode data dan tampilan.

## 2. Strategi dan Cara Mengatasinya

Untuk mengatasi hambatan tersebut, saya menerapkan beberapa langkah:

- Melakukan audit riwayat git secara rutin menggunakan perintah log untuk melihat visualisasi pohon repositori.
- Melakukan komunikasi intensif dengan tim sebelum melakukan integrasi agar dokumen API Contract selalu sinkron.
- Memastikan pemisahan folder models/, views/, dan controllers/ bersih sejak awal proyek.
